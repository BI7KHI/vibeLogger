# vibeLogger
**Vibe your logging experience!**

## 项目特点

- **极速敲录**：常用 QTH、设备、天线、功率全部支持数字编号快捷输入。点名时只需敲"序号+回车"，告别繁琐打字。

- **自动进化**：具备"自学习"功能。遇到新呼号、新地点、新设备，直接输入一次，程序自动存入联想库，下次再见即可序号直选。

- **智能自动化**：自动强制呼号大写，自动记录 HH:mm 时间戳，自动生成递增序号。

- **安全可靠**：数据实时写入 Excel和CSV，每录入一条存盘一次，不怕断电或误操作。

- **离线轻量**：一个脚本、一个 JSON 配置文件即可运行，极其适合野外架台或比赛现场使用。

**一句话总结**：它把原本手忙脚乱的日志记录变成了简单的"选数字"游戏

## 版本说明

### GUI版本 (推荐)
- **文件**：`VibeLogger_gui.py` 
- **可执行文件**：`dist/VibeLogger_GUI.exe` (29MB)
- **特点**：图形界面 + 命令行终端双模式

### 命令行版本
- **文件**：`VibeLogger.py`
- **特点**：纯命令行操作，极致轻量

## 快速开始

### 方式1：使用可执行文件 (无需Python环境)
1. 下载 `dist/VibeLogger_GUI.exe`
2. 双击运行或使用 `启动VibeLogger.bat`
3. 程序会自动创建配置文件和Excel文件

### 方式2：运行Python源码
1. 安装依赖：`pip install -r requirements.txt`
2. 运行GUI版：`python VibeLogger_gui.py`
3. 运行命令行版：`python VibeLogger.py`

## GUI版本操作说明

### 主界面布局
- **左侧**：表单录入区域
  - 支持下拉选择和智能匹配
  - QTH支持拼音简拼（gz→广州，sz→深圳等）
  - 自动学习新词汇功能
  
- **右侧**：命令行终端区域
  - 黑底绿字，经典终端风格
  - 支持完整命令行录入流程
  - 实时显示操作结果

- **底部**：日志表格显示
  - 实时显示已录入的记录
  - 支持滚动查看历史数据

### 双模式录入

#### GUI表单录入
1. 在左侧表单中填入信息
2. 点击"保存当前记录"按钮
3. 数据自动保存并清空表单

#### 命令行录入
1. 在右侧终端输入 `log` 进入录入模式
2. 按提示步骤输入各字段信息
3. 支持序号快选、智能匹配、拼音简拼
4. 按 `Ctrl+C` 可随时退出录入模式

### 终端命令

```
help        - 显示帮助信息
status      - 显示当前状态  
count       - 显示记录总数
list [n]    - 显示最近n条记录（默认5条）
clear       - 清空终端
save        - 保存GUI表单记录
reset       - 重置输入表单
log         - 进入命令行录入模式

快捷键:
Ctrl+C      - 退出录入模式或清空输入
```

### 智能匹配功能

#### QTH地点匹配
- **拼音简拼**：`gz` → 广州，`sz` → 深圳，`lg` → 龙岗
- **关键词匹配**：`广` → 广州，`深` → 深圳
- **序号快选**：`1` → 第一个选项

#### 设备/功率/天馈匹配
- **关键词匹配**：`uv` → UV-K5, UV-K6
- **序号快选**：`1` → 第一个选项
- **直接输入**：支持输入新内容，自动学习

### 自动学习功能
- 输入新的QTH、设备、功率、天馈信息时
- 程序自动添加到配置文件
- 下次使用时可通过序号或匹配选择
- 配置保存在 `log_config.json` 文件中
