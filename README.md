# vibeLogger
**Vibe your logging experience!**
- 极速敲录：常用 QTH、设备、天线、功率全部支持数字编号快捷输入。点名时只需敲“序号+回车”，告别繁琐打字。

- 自动进化：具备“自学习”功能。遇到新呼号、新地点、新设备，直接输入一次，程序自动存入联想库，下次再见即可序号直选。

- 智能自动化：自动强制呼号大写，自动记录 HH:mm 时间戳，自动生成递增序号。

- 安全可靠：数据实时写入 Excel，每录入一条存盘一次，不怕断电或误操作。* 

- 离线轻量：一个脚本、一个 JSON 配置文件即可运行，极其适合野外架台或比赛现场使用。

一句话总结：它把原本手忙脚乱的日志记录变成了简单的“选数字”游戏
